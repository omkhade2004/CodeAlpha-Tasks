# Exploratory Data Analysis (EDA)

## Objective
Perform exploratory data analysis to understand the structure,
patterns, and data quality of the dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Steps Performed
1. Data loading and inspection
2. Data type correction
3. Missing value and duplicate checks
4. Univariate and multivariate analysis
5. Correlation and outlier detection

## Conclusion
The dataset was successfully explored, cleaned, and analyzed
to identify patterns, relationships, and potential issues.
